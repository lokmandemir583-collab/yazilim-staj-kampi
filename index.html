Yazılım Staj Kampı

Bu sayfa, yazılım mühendisliği stajına hazırlık sürecimde öğrendiğim konuları takip etmek için oluşturuldu.

Öğreneceğim Konular

  • Git ve GitHub kullanımı
  • HTML ve CSS temelleri
  • JavaScript temelleri
  • API mantığı
  • Basit proje geliştirme

Hedefim

Staja başladığımda verilen projeleri anlayabilmek, küçük hataları çözebilmek ve ekibe katkı sağlayabilecek seviyeye gelmek.

Bugün Öğrendiklerim

  1. HTML dosyasının sayfa iskeletini oluşturduğunu öğrendim.
  2. CSS dosyasının sayfanın görünümünü düzenlediğini öğrendim.
  3. HTML ve CSS dosyalarını birbirine bağlamayı öğrendim.
  4. Yaptığım değişiklikleri GitHub’a göndermeyi tekrar ettim.
